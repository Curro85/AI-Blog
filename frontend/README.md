# The Frontend

### Components:
- `ArticleDetail.jsx` this component holds the full article which is only shown when you click on read more.
- `ArticleCard.jsx` a simple way to keep all articles organized creating a card for everyone.
- `ArticleList.jsx` here is a list which holds all article cards to show them in a grid.
- `Blog.jsx` the main component which contains the list and calls the backend to get the articles which goes to the other components.

### Styling:
- Using TailwindCSS to make the style of the components, creating a simple blog view.
